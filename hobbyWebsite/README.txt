misschien is de projectnaam hobbywebsite misschien een wat vreemde naam voor dit project, maar ik wil hier eigenlijk een serieuze website van maken die ik zelf later kan gebruiken

Pagina indeling:
-Main
	-Overheidsdiensten
	-Admin
		- speler informatie
		- logs
	- blog
		- ...
	- product koersen
...

Header: iedere pagina heeft een header waar je dus makkelijk van pagina naar pagina kan gaan.
ook moet er een manier komen om een account aan te maken op de site

Main: een pagina waar je op komt als je de site bezoekt. hier komen onder andere basis info op te staan.
dit zijn bijvoorbeeld de aantal spelers op de server, de recentste blog.

Overheidsdiensten: dit is voor de mensen die Politie zijn in mijn roleplay server. deze mensen kunnen dan kijken naar informatie zoals de strafbare feiten die spelers
hebben gedaan op de server. ook kunnen ze daar verklaringen geven ect

Admin: deze pagina moet alleen zichtbaar zijn voor de mensen met hogere rechten. normale mensen moeten hier niet op komen. zelfde geld voor overheidsdiensten
dat normale "burgers" niet in in deze informatie kunnen kijken.

blog: natuurlijk wil ik graag mijn updates delen op mijn site waar ik dus dingen kan bijhouden van de laatste stand van zaken

product koersen: ik heb een script geschreven dat prijzen van ingame goederen muteerd in de loop van de tijd. dit zou ik graag ook op de site neerzetten natuurlijk.
met een API kan ik dit dus heel mooi en keurig neerzetten
